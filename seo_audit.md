# SEO 종합 검토 및 개선 사항

## 현재 SEO 상태 분석

### ✅ 이미 잘 구현된 항목:
1. **메타 태그**
   - title, description, keywords 설정됨
   - Open Graph (Facebook) 태그
   - Twitter Card 태그
   
2. **구조화된 데이터 (Schema.org)**
   - HairSalon LocalBusiness 스키마
   - 영업시간, 주소, 전화번호
   - 리뷰 평점 정보
   - 서비스 목록

3. **다국어 지원**
   - og:locale 설정 (ko_KR, en_US, ja_JP)

4. **사이트맵 & Robots.txt**
   - sitemap.xml 생성됨
   - robots.txt 설정됨

---

## 🔧 개선이 필요한 항목:

### 1. 메타 태그 개선
- [ ] Canonical URL 추가 필요
- [ ] hreflang 태그 추가 (다국어 SEO)
- [ ] 도메인 URL 업데이트 (leekaja-myeongdong.com → myeongdonghairsalon.co.kr)
- [ ] favicon 및 Apple touch icon 추가
- [ ] theme-color 메타 태그 추가

### 2. 구조화된 데이터 확장
- [ ] FAQ Schema 추가
- [ ] BreadcrumbList Schema 추가
- [ ] Service Schema 확장
- [ ] Review Schema 개별 리뷰 추가

### 3. 이미지 최적화
- [ ] 모든 이미지에 alt 속성 확인
- [ ] OG 이미지 실제 파일 확인 필요
- [ ] 이미지 lazy loading 확인

### 4. 성능 최적화
- [ ] 프리로드/프리커넥트 리소스 힌트
- [ ] CSS/JS 최적화
- [ ] 폰트 최적화

### 5. 접근성
- [ ] aria-label 추가
- [ ] 시맨틱 HTML 구조 개선
- [ ] 색상 대비 확인

### 6. 소셜 미디어
- [ ] 실제 SNS 링크 확인 및 업데이트

---

## 🎯 우선순위 높은 개선 사항:

1. **Canonical URL 및 hreflang 태그** (SEO 핵심)
2. **도메인 URL 업데이트** (현재 도메인과 일치)
3. **FAQ Schema** (검색 결과 향상)
4. **favicon 추가** (브랜드 신뢰도)
5. **이미지 최적화** (페이지 속도)
